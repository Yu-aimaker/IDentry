IDentry デモ発表

信頼されるデジタルポートフォリオを
誰でも簡単・楽しく作成できるWebサービス
📝

プロジェクト概要

  • 10ステップの直感的な入力フロー
  • モダンで洗練されたUI/UX
  • Google・メール認証対応
  • リアルタイム同期・安全なデータ管理
💻

技術スタック

  • Next.js 14（App Router）
  • Tailwind CSS（モダンUI）
  • Supabase（BaaS, 認証, DB）
  • TypeScript, React Hooks, PostgreSQL
🔄

ユーザーフロー

  1. LP（ランディングページ）
  2. 10ステップでプロフィール作成
  3. プレビュー画面で確認
  4. Google/メールでログイン
  5. ダッシュボードで管理・公開
[LP] → [入力] → [プレビュー] → [ログイン] → [ダッシュボード]
🗄️

データベース設計

  • profiles（プロフィール情報）
  • education（学歴）
  • career（職歴）
  • portfolio（ポートフォリオ）
Row Level Securityで安全に管理
🎨
